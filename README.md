## main

![main](./docs/main.gif)

The sandbox currently has makings of a space shooter like an asteroids clone. To get started:

```bash
go get github.com/goshlanguage/bento
cd ~/go/src/github.com/goshlanguage/bento

go build main.go
./main
```
